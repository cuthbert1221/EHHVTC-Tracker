# EHHVTC-Tracker

## Make sure to uninstall all EHH Programs first:

1. Goto Windows search
2. Type "Add or remove programs", then hit enter
3. Search for "EHH"
4. Uninstall any programs matching EHH
5. Then type "ETS2 Job logger", and uninstall
